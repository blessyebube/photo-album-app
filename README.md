# Photo Album App

This is a simple photo album application that allows users to organize and view their photos.

## Features
- Upload photos
- View photos
- Organize into albums

## Installation

1. Clone the repository.
   
   ```bash
   git clone https://github.com/blessyebube/photo-album-app.git
   cd photo-album-app
   ```

2. Install dependencies. 
   
   ```bash
   npm install
   ```
   
3. Start the application. 
   
   ```bash
   npm start
   ```
